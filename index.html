날씨 위젯
실제로 보일 요소만 켜고 끄도록 날씨 관련 세부 옵션을 한곳에 모았습니다.
날씨 위치는 학교 설정을 그대로 사용합니다. 기본 서버를 쓰거나, 필요하실 때만 개인 API 키를 이 PC에 저장해 직접 조회할 수 있습니다.
기본 서버 또는 개인 API 키 기준 연결 결과가 여기에 표시됩니다.
급식 위젯
오늘/내일 화면 밀도와 영양 정보 표시 여부를 따로 조절합니다.
바로가기 위젯
폴더와 파일을 아이콘처럼 눌러 바로 실행하는 전용 위젯입니다.
개별 폴더/파일 경로와 아이콘은 위젯 본문에서 + 버튼으로 추가하거나 수정합니다.
기록 · 도구 위젯
세부 옵션이 많지 않은 위젯은 한 곳에서 표시 여부를 관리하도록 묶었습니다.
이 묶음은 표시 여부 위주로 관리하고, 각 위젯 내부 데이터는 본문 위젯에서 직접 편집하는 흐름을 유지합니다.
🏫 학교 데이터 연결
학교 검색, 학사일정, 급식, 시간표는 기본 학교 서버를 통해 불러옵니다. 일반 사용자는 NEIS API 키를 따로 발급받지 않으셔도 됩니다.
운영자가 별도 프록시 설정을 바꾸는 경우에만 고급 설정이 필요합니다.
🌤️ 날씨 연결
기본 서버는 설치 직후 바로 사용할 수 있습니다. 다만 공용 서버는 응답 지연이나 사용량 보호의 영향을 받을 수 있어, 가장 빠르고 안정적인 사용은 개인 OpenWeather API 키를 권장합니다.
개인 키를 입력하면 이 PC에만 로컬 저장되고 OpenWeather에 직접 연결합니다. 공용 서버 사용량 보호의 영향을 받지 않으므로 실사용 기준으로는 개인 키 쪽이 더 유리합니다.
🗓️ Google Workspace 연동
로그인 버튼으로 Google 일정과 할 일을 연결할 수 있습니다. 연결 후에는 상태 카드에서 결과를 바로 확인하실 수 있습니다.
일반 사용자는 이 항목을 직접 입력하실 필요가 없습니다. 로그인 버튼으로 연결이 되지 않을 때만 별도 안내에 따라 확인해 주세요.
