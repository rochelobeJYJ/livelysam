## 1.1.6 변경 사항

### 배경화면 (멀티모니터)
- 주 모니터와 배율(DPI)이 다른 보조 모니터에서 배경화면이 어긋나거나 화면 밖으로 밀리던 문제를 수정했습니다. 이제 각 모니터의 실제 배율에 맞춰 정확히 배치됩니다.
- 저장해 둔 모니터가 연결되어 있지 않을 때(예: 외장 모니터 분리) 배경 실행이 실패하지 않고 주 모니터로 자동 전환됩니다.

### 안정성
- WebView2 실행 파일 경로를 고정값이 아니라 동적으로 탐색하도록 바꿔, 연동된 다른 프로그램이 바뀌어도 배경화면이 실행되도록 했습니다.
- 런처 중복 실행을 방지해(단일 인스턴스) 불필요한 백그라운드 프로세스가 쌓이지 않도록 했습니다.

## 1.1.5 변경 사항

### 보안
- 로컬 브리지의 `/health` 엔드포인트가 신뢰할 수 없는 출처(샌드박스 iframe 등)에 인증 토큰을 노출하지 않도록 차단했습니다.
- HTML 이스케이프가 따옴표까지 처리하도록 보강해, 동기화된 Google 일정 제목 등을 통한 속성 주입을 방지했습니다.
- Google 재로그인 시 이미 승인한 권한에 대해 동의 화면을 다시 요구하지 않도록 스코프 비교를 정정했습니다.

### 성능
- 달력이 렌더마다 Google 일정 캐시를 셀 단위로 복제하던 것을 날짜 색인 1회 생성으로 개선했습니다.
- 위젯을 병렬로 초기화해 느린 네트워크에서도 시작이 지연되지 않도록 했습니다.
- 달 위상 렌더를 매초에서 날짜 변경 시로 줄였습니다.
- 엑셀 라이브러리를 버전 고정 후 실제 사용 시에만 지연 로드합니다.

### 안정성
- 로컬 브리지 클라이언트 로직을 단일 모듈로 통합하고, 토큰 회전 시 403 자동 복구를 모든 로컬 요청 경로에 적용했습니다.
- 아날로그 시계가 테마 색상을 반영하지 못하던 문제를 수정했습니다.

## 설치 안내

LivelySam 설치 파일은 현재 코드서명 없이 배포되고 있어, 일부 브라우저나 Windows에서 경고 메시지가 표시될 수 있습니다.
공식 GitHub Releases 페이지에서 다운로드한 파일이라면 정상 파일입니다.

설치가 막히는 경우에는 아래 순서로 진행해 주십시오.

1. 설치 파일(`LivelySamSetup-버전.exe`)을 다운로드합니다.
2. 브라우저가 다운로드를 차단하면 `유지`, `계속 다운로드`, `그래도 계속` 가운데 표시되는 항목을 선택합니다.
3. Windows에서 "PC를 보호했습니다" 또는 유사한 경고가 나오면 `추가 정보`를 누른 뒤 `실행`을 선택합니다.
4. 설치를 완료한 뒤 LivelySam을 실행합니다.

공식 배포 주소:
[https://github.com/rochelobeJYJ/livelysam/releases](https://github.com/rochelobeJYJ/livelysam/releases)

## 설치 FAQ

### 브라우저가 위험한 파일처럼 표시합니다.

현재 설치 파일에는 코드서명이 적용되지 않아, 브라우저에서 경고가 표시될 수 있습니다.
공식 GitHub Releases 페이지에서 받은 파일이라면 정상 파일입니다.

### Edge에서 다운로드가 차단됩니다.

다운로드 목록에서 `유지`를 누른 뒤, 추가 안내가 나오면 `그래도 계속` 또는 `계속 다운로드`를 선택해 주십시오.

### Windows에서 "PC를 보호했습니다"라고 나옵니다.

`추가 정보`를 누르면 `실행` 버튼이 나타납니다.
공식 GitHub Releases에서 받은 설치 파일이 맞다면 `실행`을 눌러 설치를 진행하시면 됩니다.

### 설치 파일에 개인 데이터가 들어 있나요?

아닙니다. 설치 파일에는 사용자의 로컬 데이터가 포함되지 않습니다.

### 업데이트 후에도 같은 경고가 다시 나올 수 있나요?

그럴 수 있습니다. 새 버전 설치 파일은 별도의 새 파일로 인식될 수 있어, 브라우저나 Windows에서 다시 경고를 표시할 수 있습니다.

공식 GitHub Releases 외부 링크나 메신저 전달본으로 배포된 파일은 사용하지 않는 것을 권장합니다.
