## 설치 안내

LivelySam 설치 파일은 현재 코드서명 없이 배포되고 있어, 일부 브라우저나 Windows에서 경고 메시지가 표시될 수 있습니다.
공식 GitHub Releases 페이지에서 다운로드한 파일이라면 정상 파일입니다.

설치가 막히는 경우에는 아래 순서로 진행해 주십시오.

1. 설치 파일(`LivelySamSetup-버전.exe`)을 다운로드합니다.
2. 브라우저가 다운로드를 차단하면 `유지`, `계속 다운로드`, `그래도 계속` 가운데 표시되는 항목을 선택합니다.
3. Windows에서 "PC를 보호했습니다" 또는 유사한 경고가 나오면 `추가 정보`를 누른 뒤 `실행`을 선택합니다.
4. 설치를 완료한 뒤 LivelySam을 실행합니다.

공식 배포 주소:
[https://github.com/rochelobeJYJ/livelysam/releases](https://github.com/rochelobeJYJ/livelysam/releases)

## 설치 FAQ

### 브라우저가 위험한 파일처럼 표시합니다.

현재 설치 파일에는 코드서명이 적용되지 않아, 브라우저에서 경고가 표시될 수 있습니다.
공식 GitHub Releases 페이지에서 받은 파일이라면 정상 파일입니다.

### Edge에서 다운로드가 차단됩니다.

다운로드 목록에서 `유지`를 누른 뒤, 추가 안내가 나오면 `그래도 계속` 또는 `계속 다운로드`를 선택해 주십시오.

### Windows에서 "PC를 보호했습니다"라고 나옵니다.

`추가 정보`를 누르면 `실행` 버튼이 나타납니다.
공식 GitHub Releases에서 받은 설치 파일이 맞다면 `실행`을 눌러 설치를 진행하시면 됩니다.

### 설치 파일에 개인 데이터가 들어 있나요?

아닙니다. 설치 파일에는 사용자의 로컬 데이터가 포함되지 않습니다.

### 업데이트 후에도 같은 경고가 다시 나올 수 있나요?

그럴 수 있습니다. 새 버전 설치 파일은 별도의 새 파일로 인식될 수 있어, 브라우저나 Windows에서 다시 경고를 표시할 수 있습니다.
