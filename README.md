# LivelySam

LivelySam은 Windows용 로컬 대시보드형 배경 실행기입니다.  
학교 일정, 시간표, 급식, 날씨, 메모, 할 일, Google 일정, 로컬 바로가기 등을 하나의 화면에 배치해 사용할 수 있습니다.

기본 실행은 전용 런처를 기준으로 합니다. 브라우저 미리보기 모드도 지원하지만, 일부 로컬 기능은 제한될 수 있습니다.

## 주요 기능

- Windows 설치 파일 기반 배포
- 전용 런처를 통한 실행, 중지, 로그 확인, 업데이트 확인
- GitHub Releases 기반 업데이트 및 직접 다운로드 경로 제공
- 위젯 드래그 이동, 크기 조절, 표시/숨김, 레이아웃 저장
- 학교 검색 기반 학교 코드/주소 연결
- NEIS 기반 시간표, 급식, 학사일정 연동
- Google Calendar / Google Tasks 동기화
- 학교 기준 날씨/미세먼지/예보 표시
- 즐겨찾기, 로컬 파일/폴더 바로가기, 메모, 할 일, D-Day, 타이머 제공
- 미니게임 허브 및 명예의 전당 기능 포함

## 위젯과 화면 구성

LivelySam은 고정된 템플릿이 아니라, 사용자가 직접 화면을 조정하는 방식으로 설계되어 있습니다.

- 위젯 재배치
- 위젯 크기 조절
- 위젯 표시/숨김
- 위젯 투명도 조절
- 레이아웃 저장 및 복원
- 편집 모드 외 예기치 않은 레이아웃 변경 방지

기본 위젯 구성:

- 시계(오늘 달의 위상 포함)
- 시간표
- 캘린더
- 날씨
- 급식
- 타이머
- D-Day
- 메모
- 할 일
- 즐겨찾기
- 바로가기

## 학교 등록

학교 관련 기능은 학교명 입력에서 시작합니다.

- 학교명을 입력하면 NEIS 기준 학교 검색으로 학교명, 교육청 코드, 학교 코드, 주소를 연결합니다.
- 연결된 학교 정보는 급식, 학사일정, 시간표, 날씨 기준 위치에 함께 사용됩니다.
- 학교를 다시 변경하면 관련 기능도 같은 기준으로 갱신됩니다.

## 시간표 등록

시간표는 두 가지 방식으로 사용할 수 있습니다.

학급 시간표 모드:

- 학년과 반을 설정하면 NEIS 주간 시간표를 기준으로 자동 연동합니다.

교사용 시간표 모드:

- 교과 담당 또는 개별 수업표 관리가 필요한 경우 직접 입력 방식으로 사용할 수 있습니다.
- 요일별/교시별 입력을 지원합니다.
- 교사용 시간표 양식 다운로드와 Excel 업로드를 지원합니다.

추가 보정 기능:

- 일회성 보강
- 주간 보정
- 시험 시간표
- 과목별 색상 지정

## 급식 정보

급식 위젯은 다음 구성을 지원합니다.

- 오늘 급식
- 내일 급식
- 주간 급식 보기
- 중식/석식 구분
- 칼로리 및 알레르기 정보 표시 옵션
- 간단 모드와 상세 모드 전환
- 이번 주 급식 요약
- 급식이 없는 요일 표시

## 캘린더와 Google 동기화

Google Workspace 연동을 통해 로컬 화면과 Google 일정을 함께 사용할 수 있습니다.

- Google 로그인 연결
- Google Calendar 동기화
- Google Tasks 동기화
- 사용할 캘린더와 작업 목록 선택
- 연결 상태 및 동기화 상태 확인

캘린더에는 학교 일정, Google 일정, 할 일, D-Day, 천문 이벤트를 함께 표시할 수 있습니다.

## 날씨 기능

날씨 위젯은 다음 정보를 지원합니다.

- 현재 날씨
- 체감 온도와 상세 날씨
- 시간별 예보
- 날짜별 예보
- 현재 미세먼지
- 시간별/날짜별 미세먼지 예보
- 날씨 경고 메시지
- 오늘 밤/이번 주 천문 볼거리

지원 방식:

- 기본 날씨 서버 사용
- 개인 OpenWeather API 키 사용 가능
  (더 나은 성능을 위한다면 개인 API키를 발급받아 사용하세요.(무료))

위치 기준:

- 학교 주소 기준

## 즐겨찾기와 바로가기

즐겨찾기 위젯:

- 웹 링크 저장
- 상단 고정
- 검색
- 보관/복원
- 태그 및 설명 관리

바로가기 위젯:

- 로컬 파일/폴더 등록
- 드래그 앤 드롭 추가
- 직접 경로 입력 등록
- 아이콘 및 강조 색상 설정
- 라벨/경로 표시 옵션

## 기타 기능

- 메모
- 할 일
- D-Day
- 타이머
- 미니게임 허브
- 명예의 전당

미니게임 허브에는 Dino Run, Wing Tap, OceanTap, Parcel Stack, Lane Dash 계열 게임이 포함되어 있습니다.

## 사용 흐름

### 1. 학교 등록

1. 설정에서 학교명을 입력합니다.
2. 학교 검색으로 정확한 학교를 선택합니다.
3. 학교 코드와 주소가 연결되면 급식, 학사일정, 시간표, 날씨 기준이 함께 준비됩니다.

### 2. 시간표 등록

1. 학년과 반을 선택합니다.
2. 시간표 방식을 고릅니다.
3. 학급 자동 연동 모드면 NEIS 시간표를 불러오고, 교사용 모드면 직접 입력 또는 Excel 업로드를 사용합니다.
4. 필요하면 보강, 주간 보정, 시험 시간표를 추가로 적용합니다.

### 3. Google 동기화

1. Google 로그인 후 동기화할 수 있습니다.

### 4. 날씨 설정

1. 기본 모드를 사용하거나 OpenWeather API 키를 입력합니다.
2. 학교 주소를 기준 위치로 사용합니다.
3. 현재 날씨, 예보, 미세먼지 표시 옵션을 조정합니다.

## 설치 및 업데이트

- GitHub Releases에서 최신 설치 파일을 배포합니다.
- 설치 후에는 런처에서 업데이트 확인이 가능합니다.
- 자동 업데이트 경로와 별도로 직접 다운로드 링크도 제공합니다.
- 새 설치 사용자는 최신 설치 파일을 받게 되며, 기존 사용자는 런처에서 최신 버전 여부를 확인할 수 있습니다.
- 기본 실행 파일은 `dist\launcher\LivelySamLauncher.exe` 입니다.

## 데이터 저장

LivelySam은 기본적으로 로컬 우선 방식으로 동작합니다.

- 사용자 설정과 데이터는 `%LocalAppData%\LivelySam` 아래에 저장됩니다.
- 공유 저장소 파일은 `%LocalAppData%\LivelySam\user-data\shared-storage.json` 경로를 사용합니다.
- 설치 파일에는 사용자의 로컬 데이터가 포함되지 않습니다.
- 학교 정보, 날씨, Google 동기화는 필요한 범위에서만 외부 API 또는 연결 서비스를 사용합니다.
